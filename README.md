# TikFusion-Style Telegram Repurpose Bot

This bot downloads a video and generates multiple slightly-modified variations (zoom, brightness, saturation, audio tempo, CRF, preset, etc.) â€” useful for repurposing and reposting content.

Click below to deploy to Render:

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

After deployment, go to **Environment Variables** and add:

- `BOT_TOKEN` = your Telegram bot token
- `NUM_VARIATIONS` = (optional) how many versions to create

Send any video or URL to your bot to generate variations.
